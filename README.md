# 💰 CashGuru

O **CashGuru** é um gerenciador financeiro pessoal desenvolvido para consolidar meus estudos em arquitetura de software, padrões de projeto e desenvolvimento Full Stack.

O projeto nasceu da necessidade de uma ferramenta simples, porém robusta, para controle de gastos e planejamento financeiro.

O protótipo e a identidade visual foram planejados no Figma:

## 🛠️ Tecnologias e Conceitos

- **Backend:** Node.js com NestJS (TypeScript).
- **Frontend:** React / Next.js (ou a tech que você estiver usando).
- **Arquitetura:** - Aplicação do **Strategy Pattern** para diferentes tipos de processamento financeiro.
  - Encapsulamento estrito com parâmetros privados.
  - Estrutura de **Monorepo**.

## 📁 Estrutura do Repositório

- `/cashguru-backend`: API REST construída com NestJS.
- `/cashguru-frontend`: Interface web para interação do usuário.

## 🚀 Como executar

### Backend

1. Entre na pasta `cashguru-backend`.
2. Instale as dependências: `npm install`.
3. Inicie o servidor: `npm run start:dev`.

### Frontend

1. Entre na pasta `cashguru-frontend`.
2. Instale as dependências: `npm install`.
3. Inicie o projeto: `npm run dev`.
